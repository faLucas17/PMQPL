# Gestion de Projet

Ce projet est une application de gestion de projet simple en Python.

## Installation

1. Cloner le dépôt :
    ```bash
    git clone <url_du_depot>
    cd projet_gestion
    ```

2. Installer les dépendances :
    ```bash
    pip install -r requirements.txt
    ```

## Utilisation

Pour exécuter le programme principal :
```bash
python main.py
